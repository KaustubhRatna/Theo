# Theo Research: AI vs Human Image Detection

> A research repository investigating robust AI-generated image detection through classical computer vision, deep learning, explainability, and cross-generator generalization.

---

## Overview

The rapid advancement of generative AI models such as Stable Diffusion, Midjourney, DALL·E, FLUX, and others has made distinguishing synthetic images from authentic photographs increasingly difficult. While many existing detectors achieve high accuracy on individual datasets, their performance often deteriorates when evaluated on images generated by previously unseen models.

This repository aims to study **generalizable AI image detection** rather than optimizing performance on a single benchmark.

The project follows a reproducible research methodology, beginning with classical machine learning baselines and progressing toward deep learning, multi-dataset training, and ensemble methods.

---

## Research Goals

Our primary objectives are:

- Develop robust AI vs Human image detectors.
- Study cross-generator generalization.
- Benchmark classical computer vision against modern deep learning.
- Understand *why* detectors succeed or fail through explainability.
- Build a unified evaluation framework for future research.
- Create reproducible experiments suitable for academic publication.

---

## Research Questions

The project investigates questions such as:

- Which visual cues best distinguish AI-generated images?
- How well do handcrafted features compare with deep neural networks?
- Which architectures generalize best to unseen image generators?
- Can multiple complementary detectors improve robustness?
- What failure modes exist in current AI image detectors?
- How reliable are model confidence estimates?
- Can explainability reveal meaningful forensic cues?

# Research Roadmap

## Experiment A — Classical Computer Vision Baseline

Objective:

Establish a reproducible handcrafted-feature baseline using traditional image processing techniques.

Features:

- Raw RGB
- Color Histograms
- Discrete Cosine Transform (DCT)
- Histogram of Oriented Gradients (HOG)
- Local Binary Patterns (LBP)
- Gray-Level Co-occurrence Matrix (GLCM)
- Wavelet Features

Models:

- LightGBM
- XGBoost

Outputs:

- Feature Importance
- SHAP Analysis
- Calibration
- Error Analysis

# Standard Experimental Protocol

Every experiment follows the same protocol whenever applicable.

## Dataset

- Fixed train/validation/test split
- Stratified sampling
- Fixed random seed
- Version-controlled datasets

---

## Preprocessing

Deep Learning Experiments:

- Random Horizontal Flip
- Random Crop
- Color Jitter (light)
- Small Rotation
- Optional Random Erasing

Classical ML Experiments:

- Deterministic preprocessing only
- No augmentation
